# 🎣 Phishing Simulation using Gophish

## 📌 Description
This project simulates a phishing attack to evaluate user awareness and identify potential security risks.

## 🛠 Tools Used
- Gophish  
- Kali Linux  
- VirtualBox  

## ⚙️ What I Did
- Created a phishing email template  
- Configured phishing campaign  
- Sent emails  
- Tracked user interactions  

## 📊 Results
- Users clicked phishing links  
- Demonstrated security risks  

## 🔐 Key Learning
Learned how phishing attacks work and how to prevent them.
